## 3. Elfie App (Consumer — For Everybody)

The core free product. All features are free; Elfie earns revenue from B2B partnerships, not from users.

### 3.1 Wellness Programs

- **Scope:** 20+ doctor-reviewed programs  
- **Areas covered:** weight management, nutrition, sleep, hydration, menstrual cycles, pregnancy, stress, alcohol & smoking cessation, physical activity  
- **Extras:** personalized goals, 1,000+ articles & quizzes  
- **Policy:** no ads, no paywalled features

### 3.2 Therapeutic Areas (Chronic Disease Management)

- **400+** diseases supported  
- **200+** biomarkers tracked, **100+** symptoms logged  
- **4M+** medications in the database  
- **Not a medical device** — positioned as a companion, not a diagnostic tool  
- Primary targets: hypertension, type-2 diabetes, obesity, dyslipidemia, chronic stress, COPD, cancer, heart failure, angina

### 3.3 Data Integration

| Integration | Detail |
|---|---|
| Apple Health / Google Fit | Bidirectional sync |
| Wearable devices | 500+ devices supported |
| Lab results | Manual import & storage |
| CGM (Continuous Glucose Monitor) | Direct integration |

### 3.4 Gamification System

**Currency:** Elfie Coins — earned by tracking vitals, walking, and managing medication.  
**Total coins earned to date:** 3,185,643,097+

| Feature | Description |
|---|---|
| Rewards & charities | Redeem coins for brand vouchers (e.g. Nike), discounts, or charitable donations |
| Challenges | Health challenges between users |
| Quizzes | Health knowledge quizzes earning coins |
| Leaderboards | Rankings within the user community |
| Badges | Achievement unlocks for milestones |

### 3.5 Family Care

- One app, multiple profiles  
- Child care (toddler/child) and elderly parent management  
- Family pairing — follow a family member who has their own Elfie account  
- Two-person pregnancy mode (partner co-tracking)

### 3.6 Proactive Engagement

- The only app that proactively reaches out to patients at risk of low adherence  
- **Retention:** up to 55% after 1 year (consumer), up to 90% for insurer-sponsored users

### 3.7 Medical Endorsement

- Recommended by 1,000s of doctors, cardiologists, and endocrinologists worldwide  
- Designed by doctors, behavioral scientists, and patients

## 4. ElfieCare™ (For Healthcare Professionals)

A complete clinical workspace — free for all healthcare professionals.

### 4.1 Three Core Tools

#### Pre-Visit Agent
- Collects patient symptoms, history, and concerns before the appointment  
- Accessible via simple web link or WhatsApp  
- Accepts text, voice, photos, documents  
- Feeds directly into EHR or prints for paper-based clinics

#### Consultation Scribe
- Ambient AI + dictation + real-time clinical intelligence  
- Output templates: SOAP, DAP, or custom  
- 50+ languages  
- Works on desktop and mobile  
- Offline mode when network is unavailable  
- Clinician reviews and finalizes all output

#### Evidence AI
- Answers any clinical question with cited responses  
- 100+ scientific sources indexed hierarchically  
- Every claim links to its published source  
- Available during and after consultation

### 4.2 Learning Module

- CME modules, expert podcasts, mentor trainings  
- Filtered by country, specialty, and practice type  
- CME accreditation via local body; completion tracking via Elfie  
- Cross-device (start in clinic, finish on commute)

### 4.3 One Workspace for the Whole Team

- Supports: physicians, nurses, medical assistants, front desk  
- Role-based access control  
- EHR integration optional (or use ElfieCare as standalone EHR)  
- Embedded audit trail for compliance

### 4.4 Patient Dashboard ("Show Me Your Elfie")

- Offsite vitals and medication adherence data surfaced in under 10 seconds  
- No training required for clinical staff  
- No integration required to start  
- 100% worldwide data privacy

### 4.5 Integration Points (Healthcare)

| System | Integration Type |
|---|---|
| Pharmacy | Purchase event → automatic pillbox update |
| Appointment systems | Appointment reminder push |
| EHR | Bidirectional data sync (offline → EHR) |

### 4.6 Frontline Staff Rewards

- Reward coins for onboarding patients onto Elfie  
- Leaderboard for health challenges across nursing staff  
- Patient statistics dashboard per agent

### 4.7 Impact Metrics

| Metric | Value |
|---|---|
| Time saved | −150 h/year per doctor |
| Missed revisits | −5% |
| Time to get started | 2 minutes |
| Cost | $0 for all healthcare professionals |
